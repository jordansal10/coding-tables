colspan is used to span columns of a table while rowspan is used to span rows of a table.

Cell padding is the space between a table cell's border and content. Cellspacing is defined as the space betweeen each individual adjacent cells.

To define a table borders using CSS: 'table, th, td { border : 4px solid black }'

This tag can be used to define border width 'div {border-width: thin;}'

<table> tag is used to define a table

  • <tr> adds a table row
  • <th> adds a table header
  • <td> adds a table column or another piece of data
  • <thead> used to group table header content, used in association with <tbody> and <tfoot>
  • <tfoot> adds a table footer
  • <tcaption> adds a table caption
